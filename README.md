# 📊 Excel Data Analytics Dashboard Projects

🚀 **End-to-End Data Analysis Projects using Microsoft Excel**

📌 This repository showcases my practical experience in data analytics using Excel, where I worked on real-world datasets to extract meaningful insights and build interactive dashboards for decision-making.

---

## 🧾 Project Overview

This repository contains multiple Excel-based data analytics projects that demonstrate the complete workflow of transforming raw data into actionable insights.

Each project follows a structured approach including:
- Data cleaning and preprocessing  
- Data analysis using pivot tables  
- Data visualization through charts  
- Dashboard creation for business insights

## 📂 Projects Included

### 🏬 1. Store Data Analysis Dashboard

#### 📌 Objective:
To analyze store performance and identify key business trends.

#### 🔍 Analysis Performed:
- Cleaned and structured raw sales data  
- Analyzed product category performance  
- Compared sales across different regions  
- Identified high-performing products  

#### 📊 Dashboard Highlights:
- Sales distribution by category  
- Region-wise performance analysis  
- Trend analysis using charts  
- Interactive filters for dynamic insights  

#### 💡 Key Insights:
- Identified top-selling categories contributing to revenue  
- Found regions with highest and lowest sales  
- Highlighted opportunities for business growth
